# 283. Move Zeroes

**Difficulty:** Easy  
**LeetCode:** [Link to Problem](https://leetcode.com/problems/move-zeroes/)

## Approach
- Use two-pointer technique  
- Swap non-zero elements forward  
- Zeros automatically shift to the end  

## Complexity
- Time: O(n)  
- Space: O(1)
